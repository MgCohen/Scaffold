# com.scaffold.example (backend template)

Sample `Backend~/Scaffold/Example` + `Scaffold/Example.DTO` layout for a new LiveOps feature module. After you copy this tree into a real package, **Install or Update Backend** registers the new `.csproj` files in `LiveOps/LiveOps.Deploy.sln` under the **Scaffold/Example** solution folder (DTO and runtime projects share one folder; see `LiveOpsBackendInstall.MapCsprojToSolutionFolder` in `com.scaffold.liveops`).
